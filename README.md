# E-commerce Platform

## 🚧 Under Development 🚧
This is an E-commerce platform currently under development. The platform is a work-in-progress and is not yet fully functional. Many features are still being implemented, and the site may contain bugs or incomplete functionality. Stay tuned for updates!

## ⚙️ Features (Planned)
- **User Authentication**: Login/Registration system for users.
- **Product Catalog**: Browse products with search and filter functionality.
- **Shopping Cart**: Add items to cart and proceed to checkout.
- **Order Management**: Admin can view and manage customer orders.
- **Payment Integration**: Stripe/PayPal integration for payments.
- **User Profiles**: Manage user information and order history.
- **Reviews & Ratings**: Customers can leave reviews and rate products.
- **Admin Dashboard**: Manage products, users, and orders from the admin panel.

## 🛠️ Under Development
Currently, the following features are being developed:

> # **Note:** The platform is **not functional** until all features are implemented and tested.
